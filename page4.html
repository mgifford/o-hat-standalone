Missing Language Information

This entire page is missing the lang attribute on the <html> element.

It should have: <html lang="en">

This affects screen readers' pronunciation and text-to-speech accuracy.

Summary of Issues on This Page

  • Missing lang attribute on html element (entire page)
  • div with role="button" instead of using button element
  • span with role="link" instead of using a element
  • div with role="heading" and invalid aria-level
  • button with role="checkbox" (conflicting roles)
  • tablist with incomplete ARIA attributes (missing aria-selected, aria-controls)
